# Path to Independence (Gamer Style)

This is a simple static website built for GitHub Pages.

## Features
- Neon gamer UI
- Tabbed interface (Progress, Contract, Settings)
- Contract viewer with copy/download
- Theme selection
- Local storage for settings

## How to deploy on GitHub Pages
1. Create a repository named `path-to-independence`
2. Upload all files from this ZIP to the repo root
3. Go to **Settings â†’ Pages**
4. Set **Source** to **main branch / root**
5. Save and wait a few minutes

The site will be live at:
`https://<your-username>.github.io/path-to-independence/`
